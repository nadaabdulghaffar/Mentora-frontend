# Mentora-frontend

Front-end for Mentora built with React, TypeScript, and Vite.

## Getting started
- Install dependencies: `npm install`
- Run locally: `npm run dev`
- Production build: `npm run build`

## Project
- Tooling: Vite + TypeScript + ESLint + TailwindCSS
- Source entry: `src/main.tsx`, root app: `src/App.tsx`
- Auth flows live under `src/pages/auth/`
